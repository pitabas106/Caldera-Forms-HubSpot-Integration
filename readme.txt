Caldera Forms HubSpot Integration

== Description ==

The Caldera Forms HubSpot Integration plugin lets you add a new HubSpot Processor to Caldera form. It automatically syncs data from your Caldera form to your HubSpot CRM when the form is submitted.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Caldera Forms HubSpot Integration".
3. Click install.
4. Click activate.
